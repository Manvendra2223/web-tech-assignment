# Web Technology Lab Work

## 📌 Description

This repository contains my Web Technology practical assignments covering frontend development and Java-based web technologies.

---

## 📚 Syllabus Covered

### UNIT-I

* HTML (Lists, Tables, Images, Forms, Frames)
* CSS (Cascading Style Sheets)
* JavaScript (Basics, Objects, Dynamic HTML)
* XML (DTD, XML Schema, DOM, SAX)

### UNIT-II

* JavaBeans (Basics, Advantages, BeanInfo)
* Bound and Constrained Properties
* Persistence and Customizers
* Introduction to EJB

### UNIT-III

* Web Servers (Tomcat)
* Servlets (Lifecycle, API)
* Handling Requests & Responses
* Cookies and Session Tracking

### UNIT-IV

* JSP Basics
* JSP Lifecycle and Processing
* MVC Architecture
* Environment Setup (JDK, Tomcat)

### UNIT-V

* JSP Application Development
* Dynamic Content Generation
* Implicit Objects
* Session & Application Data Handling
* Error Handling and Debugging
